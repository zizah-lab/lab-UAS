# Program Pengelolaan Data Mahasiswa (Python)

Program ini dibuat untuk mengelola data mahasiswa sederhana menggunakan bahasa pemrograman Python.
Data yang dikelola meliputi NIM, nama, dan nilai, kemudian program akan menampilkan grade serta status kelulusan dalam bentuk tabel di console.

Program ini disusun dengan konsep pemisahan file dan class, sehingga alur program lebih terstruktur, mudah dipahami, dan mudah dikembangkan.

## Tujuan Program

Tujuan dari pembuatan program ini adalah:

Menerapkan konsep Object Oriented Programming (OOP)

Memisahkan fungsi data, proses, tampilan, dan program utama

Melatih pembuatan program yang rapi dan terstruktur

Menampilkan hasil pengolahan data dalam bentuk tabel sederhana
